# OBLE Status

Current status:

- standards repository package exists
- first standalone draft packet exists as `draft-0`
- examples, schemas, profiles, and conformance guidance are present
- local validation workflow is in place

What `draft-0` means:

- the packet is coherent enough for another implementer to read and start from
- the standard is still draft-stage, not frozen
- profile boundaries are real, but still evolving

Current strongest areas:

- core model
- lifecycle and invariants
- counterparty/open-item profile
- policy/designations profile

Still evolving:

- multi-currency profile breadth
- close/reopen interoperability posture
- conformance language for non-Heft implementations
