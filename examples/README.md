# OBLE Examples

These example payloads are small, human-readable reference fixtures for the
current OBLE draft set.

Current examples:

- `core-book.json`
- `core-accounts.json`
- `core-periods.json`
- `core-entry-posted.json`
- `counterparties.json`
- `policy-profile.json`
- `classification-profile.json`
- `dimension-profile.json`
- `budget-profile.json`
- `book-snapshot.json`
- `reversal-pair.json`
- `counterparty-open-item.json`

They are illustrative, not yet normative.

Their purpose is to make the drafts easier to implement and critique by showing
what the current JSON profile looks like in practice.
